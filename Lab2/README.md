# Machine Learning with Scikit-Learn (Lab 2)

This project demonstrates basic machine learning workflows using Scikit-Learn in Python.

## 📌 Objectives

- Load and explore datasets
- Split data into training and testing sets
- Train classification models (Decision Tree and KNN)
- Evaluate model performance
- Visualize results using confusion matrices
- Apply the workflow to a custom CSV dataset (Titanic)

## 🧠 Models Used

- Decision Tree Classifier
- K-Nearest Neighbors (KNN)

## 📊 Dataset

- Iris dataset (built-in)
- Titanic-Dataset.csv (custom CSV)

## ⚙️ Workflow

1. Data loading and exploration  
2. Data preprocessing  
3. Train/test split  
4. Model training  
5. Evaluation and comparison  

## 📈 Results

- KNN generally achieved higher accuracy than Decision Tree.
- Confusion matrices were used to analyze misclassifications.
- Feature engineering (adding new columns) can impact performance.

## 🚀 How to Run

1. Open the notebook in Google Colab or Jupyter.
2. Run all cells in order.
3. Review accuracy scores and confusion matrices.
